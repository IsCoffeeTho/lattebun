LatteBun

Welcome to LatteBun, a library that makes writing endpoints a lot less redundant.
LatteBun is NOT a web framework, it's a page framework. It allows one to write pages and then programatically combine them.

There is no specific way of doing what you want to do; so this library is not a one size fits all you have to understand your problem and the different ways of solving it. This library should only serve as a tool.

Getting Started

# bash $ bun install lattebun toastiebun
// example.ts import toastiebun from "toastiebun"; import lattebun from "lattebun"; const app = new toastiebun.server(); const page = new lattebun.templateFile(`${__dirname}/page.html`); app.get("/", (req, res) => { res.send("html").send(page.fill({ name: "World" })); }); app.listen("::", 3000);
<!-- page.html --> <html> <body> Hello, {{name}}! </body> </html>
